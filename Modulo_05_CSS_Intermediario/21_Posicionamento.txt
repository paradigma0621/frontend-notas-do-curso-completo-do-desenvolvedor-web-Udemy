Já vimos como deixarmos margens e padding para os blocos, agora vamos aprender em como deixar distâncias de posicionamento dos blocos (AP: funciona para outros elementos além de blocos div?).
AP: a propriedade position é aliada as propriedades top, right, bottom e left no sentido de elas darem as distâncias referenciais.

exemplo:
#pos_right {
	position: fixed;
	left: 20px;
}
