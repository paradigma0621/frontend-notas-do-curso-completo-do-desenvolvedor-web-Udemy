WWW: Word Wide Web
URL: Uniform Resource Locator: "localizador padrão de recursos" = endereço do site
HTTP: Hypertext Transfer Protocol = protocolo de transferência de hipertexto. Contém informações(ap:?) como: solicitante, solicitação (qual página está sendo solicitada) e servidor
HTML: Hypertext Markup Language = linguagem de marcação de texto
PHP:

páginas estáticas: páginas escritas em HTML com conteúdo fixo

páginas dinâmicas: págins HTML criadas em tempo de requisição
